# 表图标题与图注表注（论文定稿版）

## 图8A

- 图题：
  - 图8A 趋势调整型DID与基准DID的结果比较
- 图注：
  - 图中报告表4基准 DID 与加入省份线性时间趋势后的趋势调整型 DID 估计系数及 95% 置信区间。趋势调整型检验用于回应省际异质趋势可能影响主结果的识别顾虑，属于防守型稳健性检验，而非替代主模型。结果表明，采购阶段占比下降在趋势调整后仍保持显著负向，执行阶段占比上升的方向保持一致但统计强度有所减弱，综合治理质量口径未形成稳定支持。

## 图8B

- 图题：
  - 图8B 逐省剔除诊断下核心结果的稳定性
- 图注：
  - 图中展示逐一删除单个省份后重新估计所得系数分布，并以表4中的基准 DID 系数作为参考线。该检验用于判断推进结构结果是否主要由单一地区驱动。结果表明，执行阶段占比和采购阶段占比在删除任一省份后均未发生符号翻转，但统计强度对个别地区仍有一定敏感性；综合治理质量口径波动更大，因此更适合作为边界性结果处理。

## 表8

- 表题：
  - 表8 主识别防守型稳健性检验摘要
- 表注：
  - 本表汇总趋势调整型 DID、逐省剔除诊断和更保守小样本推断三类检验。所有比较均以表4的正式基准 DID 结果为 manuscript-facing baseline。趋势调整型 DID 与逐省剔除诊断用于主识别防守，wild cluster bootstrap 仅用于更保守推断下的边界说明。stack DID、cohort ATT 和动态补充识别不在本表展开，只在正文中简短交代其边界诊断作用。
