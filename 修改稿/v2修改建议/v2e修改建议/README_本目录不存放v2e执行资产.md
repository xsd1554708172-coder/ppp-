# 本目录不存放 v2e 执行资产

本目录仅用于保存“v2e 成稿之后”的外部审稿意见、人工修改建议或后续反馈材料。

此前放在本目录下的 `v2d → v2e` 执行计划、正文 patch、rerun checklist、province-year audit、implied city-slot registry、机制/文本效度可得性矩阵等，性质并不是“v2e 成稿后的外部修改建议”，而是 v2e 生成前后的执行工作包与审计资产。

这些资产已迁移至：

`修改稿/v2执行工作包/v2d_to_v2e_execution_assets`

正式 v2e 成稿仍以以下目录为准：

- `修改稿/v2说明文件/v2e/`
- `修改稿/v2修改稿留底/v2e/`

请勿再把 v2d→v2e 的施工图、执行日志、派生审计表或 rerun 中间资产放入本目录。
