# v2d_to_v2e_execution_assets 目录用途说明

本目录保存 `v2d → v2e` 的执行工作包与审计资产，不是 v2e 成稿之后的外部审稿意见。

本目录内容包括：

- v2d DID 冲刺修订报告的输入映射、任务清单、执行计划与 delivery note；
- 面向正文的 patch-style 修改建议；
- empirical rerun checklist、运行日志与结果快照；
- province-year treat_share audit、implied city-slot registry、working threshold proxy；
- 独立机制变量、文本效度审计、DID 文献整合等可得性或中间审计材料。

目录边界：

- `修改稿/v2修改建议/`：只放外部审稿意见、人工反馈、成稿后的修改建议；
- `修改稿/v2执行工作包/v2d_to_v2e_execution_assets/`：放 v2d→v2e 施工图、执行日志和派生审计资产；
- `修改稿/v2说明文件/v2e/`：放正式 v2e 交付说明、正式稿和 zip；
- `修改稿/v2修改稿留底/v2e/`：放 v2e 留底稿。

因此，本目录文件不能被解释为“v2e 已经成稿后又收到的一轮外部修改建议”。
