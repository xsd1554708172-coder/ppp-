# v2 执行工作包目录用途说明

本目录用于存放 v2 系列版本推进过程中的“执行工作包”和“审计资产”，包括：

- 从上一版修订报告提炼出的执行计划；
- 正文 patch-style 改写稿；
- rerun checklist 与运行日志；
- treatment registry / province-year audit / working proxy 数据；
- 机制变量、文本效度、文献整合等中间审计材料。

本目录不同于 `修改稿/v2修改建议/`：

- `修改稿/v2修改建议/` 仅保存外部审稿意见、人工修改建议、成稿后的反馈报告；
- `修改稿/v2执行工作包/v2d_to_v2e_execution_assets/` 保存的是 `v2d → v2e` 执行工作包，不是 v2e 成稿后的外部审稿意见。

正式 v2e 稿件和交付包仍以 `修改稿/v2说明文件/v2e/` 与 `修改稿/v2修改稿留底/v2e/` 为准。
