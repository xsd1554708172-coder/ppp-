# 脚本使用说明（最终版）

- 运行方式：`python did修改补强_round3_定点修复/scripts/run_did_reinforcement_round3_targeted.py`
- 输入来源：脚本会在当前项目根目录下自动搜索正式 V3 主面板和 5.3 基准 DID 正式结果表。
- 输出位置：所有结果统一写入 `did修改补强_round3_定点修复/` 下的 `表格`、`图形`、`正文替换文本`、`说明`、`日志` 子目录。
- 路径逻辑：脚本使用相对路径和 `pathlib` 自动定位项目根目录，不依赖个人 Windows 绝对路径。
- 运行前提：当前环境需可用 `pandas`、`numpy`、`matplotlib`、`statsmodels`、`openpyxl`。
