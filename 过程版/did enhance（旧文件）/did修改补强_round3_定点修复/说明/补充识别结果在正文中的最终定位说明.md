# 补充识别结果在正文中的最终定位说明

- Stack DID 最终定位：正文5.6短节中的补充识别主规格。
- cohort ATT 最终定位：附录或备查。
- 动态补充识别最终定位：附录诊断，不进入正文主论证。
- post-onset 强度检验最终定位：备查，不进入正文。

定位理由：执行阶段占比和采购阶段占比在补充识别框架下仍保持与基准DID一致的方向判断，但统计强度明显减弱，因此更适合被理解为方向性补充，而非实质补强。 质量型口径在补充识别框架下方向和统计强度均不稳，更不能承担主结论。 同时，cohort ATT 的解释力度取决于 leave-one-cohort-out 结果；若其增强效果主要由 2017 单一 cohort 驱动，则不宜在正文中以“更强补强”方式出现。
