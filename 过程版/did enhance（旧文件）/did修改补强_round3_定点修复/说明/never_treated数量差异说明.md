# never-treated 数量差异说明

正式 V3 主面板 full panel 中，never-treated 地区共有 3 个：新疆生产建设兵团, 海南, 青海。进入 `baseline_sample_5_3==1` 的正式基准样本后，never-treated 地区变为 2 个：海南, 青海。两者之差并非由 stack DID 窗口或控制组切换造成，而是来自上游正式样本筛选。

被排除的 never-treated 地区为：新疆生产建设兵团。
