# 补充识别样本量说明（表注与正文版）

表注版：补充识别中的观测量基于 stack/cohort 估计框架下的堆叠样本计数，因此不与基准 DID 的省年面板观测量一一对应。基准 DID 的正式有效样本为 262，本轮最终采用的 stack DID 主规格样本量为 160。不同 stack 规格的 N 之所以彼此不同，是因为控制组定义和事件窗口不同，会改变每个 cohort 可进入估计的控制观测数量。

正文版：需要说明的是，补充识别中的样本量不宜直接与基准 DID 的面板样本量横向比较。其差异来自堆叠估计框架对 cohort 与控制组的重新组织，而不是重新扩样本。当前正式主面板中 never-treated 共有 3 个地区，但进入正式基准样本后仅保留 2 个，未保留的地区为：新疆生产建设兵团。
