# 第8部分 8.1–8.3 常规稳健性（V2 四级十二类）

## 口径
- 面板：省—年 TWFE
- 固定效应：province FE + year FE
- 聚类：province
- 控制变量：与第5.3正式回归一致；滞后控制规格使用各控制变量的省内滞后一期值
- 基准 did 变量：treat_share

## 本轮稳健性规格
1. 替代处理变量：did_any
2. 替代处理变量：did_intensity
3. 滞后一期控制变量
4. 剔除极端年份：2014、2022
5. 剔除特殊地区：北京、上海、天津、重庆、中央、新疆生产建设兵团
6. 文本样本门槛：仅保留 doc_count >= 1 的省—年单元

## 结果摘要
- 推进结构结果仍然最稳：exec_share 在滞后控制、剔除极端年份、剔除特殊地区下保持正向；proc_share 在对应规格下保持负向。
- did_any 对 proc_share 为负且显著，但 did_intensity 的结果相对较弱。
- 文本样本门槛规格下推进结构结果不再显著，说明当前全样本“无文本年份按零处理”的制度口径对结果有一定影响，正文应把该结果作为谨慎性说明，而不是把门槛规格写成对主结论的推翻。
- ppp_quality_zindex 与 vfm_pass_rate 在常规稳健性中均未表现出稳定显著性。

## 对正文最稳的写法
- 可以写：主效应在替代处理变量、滞后控制和样本修剪下总体保持，尤其是执行阶段占比上升、采购阶段占比下降的推进结构结论较稳。
- 不宜写：所有质量型结果在稳健性检验中都成立。

- 本轮修订新增缺位虚拟变量控制：在 baseline TWFE 中显式加入 `text_missing` 后，`treat_share -> exec_share` 与 `treat_share -> proc_share` 的方向和显著性保持稳定；`text_missing` 自身不显著。对应结果见 `PPP_第8部分_8.1-8.3缺位虚拟变量控制结果_20260412.csv` 与工作簿新增sheet。
