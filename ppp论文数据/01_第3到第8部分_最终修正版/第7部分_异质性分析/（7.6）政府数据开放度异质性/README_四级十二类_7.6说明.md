# 第7部分 7.6 政府数据开放度异质性（V2 四级十二类）说明

## 口径
- 省—年 TWFE
- province FE + year FE
- 按 province 聚类稳健标准误
- 控制变量与第5.3正式回归一致
- 样本量：262

## 代理变量构造
本轮以处理前（2014–2015）省级 `A3_idx` 的均值作为“政府数据开放度”探索性代理。
原因是 `A3_idx` 对应“信息公开与透明协同”，与政府数据开放和可见治理最接近。
随后按中位数（10.9017）分为高/低开放度两组。

## 核心结果
### exec_share
- 交互项 p = 0.7466
- 低开放度组：0.2772，p = 0.3640
- 高开放度组：0.3961，p = 0.0023

### proc_share
- 交互项 p = 0.5347
- 低开放度组：-0.2294，p = 0.4670
- 高开放度组：-0.4622，p = 0.0003

### ppp_quality_zindex
- 交互项 p = 0.4814
- 低开放度组：1.1977，p = 0.3397
- 高开放度组：0.3373，p = 0.2912

## 建议写法
当前更稳的解释是：
- 在高开放度组中，改革对推进结构（exec_share 上升、proc_share 下降）的作用表现得更稳定；
- 但交互项本身并不显著，因此不宜写成“政府数据开放度显著放大或主导全部改革效应”。
