# 第6.5 链式机制检验（V2 四级十二类）说明

本批结果基于新版 `PPP_3.6_model_ready_panel_v2_四级十二类_实际执行版.csv`，
使用省—年 TWFE、province/year 固定效应、按 province 聚类稳健标准误，控制变量与第5.3正式回归一致。

## 核心结果
- treat_share -> A_idx：175.5750，p=0.0000
- A_idx -> B_idx：0.0328，p=0.5602
- B_idx -> C_idx：0.8578，p=0.0001
- C_idx -> D_idx：0.0934，p=0.0005

## 解释
当前结果支持“局部链式传导”：
1. 改革首先显著改变 A（数字治理接口）。
2. A -> B 不稳定，不宜写成完整强链条。
3. B -> C 与 C -> D 两段显著，说明过程治理与执行/风险治理之间存在较稳层级联动。
4. 在同时控制 A/B/C 后，treat_share 对 D_idx 的净直接效应不显著，说明 D 更像是前序治理结构变化的后续响应。

## 写作建议
正文宜写为：存在从治理接口变化到过程治理结构再到风险治理响应的部分传导迹象，其中 B→C 和 C→D 两段更稳；不宜写成完整稳定的 A→B→C→D 强链式中介。
