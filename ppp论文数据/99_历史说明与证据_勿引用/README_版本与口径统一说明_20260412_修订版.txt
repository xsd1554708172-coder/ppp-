版本与口径统一说明（修订版，2026-04-12）

一、统一口径
1. 文本主题体系：4个一级主题 + 12个二级主题 + 24个机器主题。
2. 主实证层级：省—年面板。
3. 主模型：多期 DID / TWFE。
4. 主处理变量：treat_share；did_any、did_intensity 仅作为替代处理变量/稳健性规格。
5. 事件研究：仅用于动态展示与识别边界说明，不得写成“平行趋势已成立”。
6. 第9部分：仅采用无标签泄漏版结果；应解释为项目级风险排序/识别任务，而非制度治理替代。

二、处理变量精确定义
设 i 为城市，p 为省份，t 为年份，N_pt 为省 p 在年份 t 的城市样本数。
1. post_t = 1[t>=2016]，否则为 0。
2. treat_share_pt = (1/N_pt) * Σ_i treat_i
3. did_intensity_pt = (1/N_pt) * Σ_i DID_it = post_t * treat_share_pt
4. did_any_pt = 1(Σ_i DID_it > 0) = 1(did_intensity_pt > 0)

三、文本缺位处理修正
1. 保留“无文本=0”的原变量口径，以维持省—年面板结构一致。
2. 同步新增 text_missing = 1(doc_count=0)、text_observed = 1(doc_count>=1)。
3. 正文应把“无文本=0”解释为制度表达缺位，而非简单缺失。

四、文件名一致化
原包中若干文件名仍使用 V1/V2 旧口径，但内容已是四级十二类重建版。现已统一重命名，并同步更新 README、清单、脚本引用与写作骨架。
