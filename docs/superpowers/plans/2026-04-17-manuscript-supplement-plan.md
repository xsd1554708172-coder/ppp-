# PPP Manuscript Supplement Plan

> **For agentic workers:** 本计划面向论文补全与定点修订，不要求直接重写 `.docx`。优先在补丁文件中完成段落级替换文本，再人工或脚本安全回填到主文稿。步骤使用 checkbox (`- [ ]`) 语法跟踪。

**Goal:** 基于当前主文稿、已完成的 DID 补强补丁、以及 `ppp论文数据\参考文献` 中的论文资料，系统补强论文的文献综述、理论补位、方法定位、扩展分析边界与参考文献规范性，在不改变主识别结构的前提下提升学术完整度与公共管理风格的一致性。

**Architecture:** 补全工作采用“主文稿结构锚定 + 文献功能分组 + 分章节增补 + 引文与口径统一”路线。主线仍以 `treat_share` 多期 DID/TWFE 为唯一主识别，新增内容只服务于文献补位、理论解释、变量工程合法性说明、扩展分析边界澄清和投稿前参考文献规范化。

**Tech Stack:** `.docx` 文稿、markdown 补丁、PDF 文献抽读、主文稿 `document.xml` 锚定、手工学术写作整合。
